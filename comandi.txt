1. Creare un branch chiamato “Feature/galleria” e spostarsi nel branch appena creato
    git branch Feature/galleria //Per creare il branch
    git checkout Feature/galleria //Per spostarsi nel branch creato

2. Modificare la pagina inserendo un Carousel Bootstrap funzionante (immagini a scelta)
    Copia e incolla un codice di carousel sul sito di Boostrap
    <img src="..." class="d-block w-100" alt="...">
    Inserisco nel src le immagini che voglio mostrare(Inserisco il percorso di dove si trova l'immagine), 
    mentre in alt inserisco la posizione con cui mostro le slide/immagini(First Slide, Second Slide....) 

3. Fare il commit con un messaggio “add carousel with n images”
    git commit -m "add carousel with n images"
